Pertanyaan yang Sering Ditanyakan

Berapa lama respon admin untuk pembelian rank? +

Admin biasanya merespons dalam 1-12 jam. Pastikan sudah mengirim bukti transfer dan konfirmasi via WhatsApp. Rank akan diaktifkan maksimal 24 jam setelah pembayaran dikonfirmasi.

Bagaimana cara join server Minecraft? +

1. Buka Minecraft versi 1.20-1.21.10
2. Klik "Multiplayer"
3. Klik "Add Server"
4. Masukkan IP: play.arnex.my.id
5. Klik "Done" dan join server!

Saya menemukan bug/cheater, harus lapor ke siapa? +

Langsung laporkan ke Admin Discord (Kry/Shiro) atau via form di atas dengan pilih kategori "Masalah Komunitas/Player". Sertakan screenshot/video bukti untuk proses lebih cepat.

Apakah ada grup WhatsApp untuk komunitas? +

Ya! Join grup WhatsApp komunitas di: Link Grup WhatsApp. Grup ini untuk sosialisasi dan pengumuman umum.

Rank saya habis masa berlaku, bagaimana perpanjangnya? +

Hubungi admin via WhatsApp 1-2 hari sebelum rank habis. Proses perpanjangan sama seperti pembelian baru. Harga tetap sama, tidak ada kenaikan.
